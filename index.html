PLIX

PLIX is a the most beautiful skin for steam that modernizes the whole steam client.

A new look for the platform you already know and love. plix is fresh and exciting from the moment you open Steam.

HOW TO INSTALL

  • Move plix (Light) or plix (Dark) folder to steam/skins/
  • go to FONTS folder and install all the fonts there
  • Pick an extra that you like from the EXTRAS folder and paste the layout file in resource/layout (replace the file)
  • On steam go to settings -> interface and choose the newly added skin, Steam will have to restart to apply the skin
  • Install https://github.com/PhantomGamers/SFP
  • take the libraryrootcustom.css file and paste it on steam/steamui/
  • take the friends.custom.css file and paste it on steam/clientui/
  • open steamfriendspatcher and it will apply the library skin

EXTRAS

  • Announcement button
  • Rounded border on avatar
  • Rounded border on avatar with balance
  • Rounded avatar with colored background on username
  • Rounded avatar with colored background on username with balance
